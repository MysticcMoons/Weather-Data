# Weather-Data

**Uses data on precipitation from each month, self made functions, and merge sort using random numbers**

## What It Does  
- Uses merge sort for 10 random integers.
- Shows the month with the most precipitation.
- Shows the month with the least precipitation.
- Shows the average precipitation of all months.
- Shows the number and names of the months whos precipitation was above the mean.
- Shows order of months in order from most precipitation to least.

## Why I Built It  
I have always been interested in weather apps and making relationships and numbers from it. This was to automate the process of looking at precipitation to make it usable numbers that can then be used to form relationships and later predictions from.
